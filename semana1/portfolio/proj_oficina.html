Oficina de Games

Este site foi criado para o estudio de jogos digitais Oficina de Games. O projeto exibe em uma só página conteúdos como Informações sobres os jogos desenvolvidos, Equipe, Redes sociais além de links para trailers e arquivos externos. A navegação pela página acontece através de âncoras.
