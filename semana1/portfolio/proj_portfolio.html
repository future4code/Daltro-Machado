Portfolio

Este projeto foi um exercicio da primeira semana do curso de Desenvolvedor Fullstack. O site teve que ser construído utilizando apenas HTML e CSS. O projeto abrange uma página inicial com informações sobre o profissional, uma página com suas experiências profissionais e acadêmicas e uma página com os projetos já realizados. Além disso existem no menu principal, botões de acesso às redes sociais e uma página para envio de mensagem.
Tecnologias utilizadas
  • HTML
  • CSS
  • Editoração Eletrônica
  • Git
